# Arun Teja Reddy Kallam — Portfolio

Personal portfolio website built with Next.js + TypeScript and deployed on Vercel.

Live: https://arunkallam.vercel.app

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Vercel (Deployment)

## Features
- Clean, section-based layout (Profile, Education, Skills, Experience, Projects, Contact)
- Smooth scrolling + active nav state
- Resume PDF link
- Responsive UI (desktop + mobile)
